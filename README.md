Weather App

A simple and responsive weather web app built with HTML, CSS, and JavaScript. Enter any city name to get live weather data including temperature, humidity, wind speed, and dynamic weather icons — all powered by the OpenWeatherMap API.

Features

Real-time weather data by city name
Dynamic weather icons based on conditions (Rain, Clouds, Clear, etc.)
Animated video background
Clean, minimal UI
Tech Stack

HTML, CSS, JavaScript
OpenWeatherMap API
Setup

Clone the repo
Add your OpenWeatherMap API key to index.html
Open with Live Server or any local HTTP serve
